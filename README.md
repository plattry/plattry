# Plattry - The plattry PHP framework

Plattry/plattry is a lightweight Web framework and an example of the application of the plattry package.

_Welcome to __join us__ or __star us__ for encouragement._

### Requirements
- Plattry/plattry works with PHP __8.0__ or above.

### Installation
Install the latest version with
```shell
$ composer require plattry/plattry
```

### Bugs and Feature
__Bugs__ and __feature__ request are tracked on __[Issue](https://github.com/plattry/plattry/issues)__.
